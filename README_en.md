## Description

Platform used for collecting and sharing testing result. Powered by [Tornado](http://www.tornadoweb.org/en/stable/index.html).

## Environment

- Python 3.5
- Ubuntu 16.04
- MongoDB 3.6

## How to

1. Install requirement:
```
pip install -r requirement.txt
```

2. Run server:
```
python index.py
```
